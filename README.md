# Two-Factor Authentication System

A secure web-based two-factor authentication system balancing robust security 
with excellent user experience.

## Features
- Password + TOTP-based authentication
- QR code setup for authenticator apps
- Secure recovery code system
- Rate limiting and brute force protection
- Encrypted credential storage
- CSRF protection

## Technologies
- Backend: Node.js, Express.js, MongoDB
- Frontend: React, HTML5, CSS3
- Security: JWT, Speakeasy, Bcrypt, Argon2, Helmet.js
